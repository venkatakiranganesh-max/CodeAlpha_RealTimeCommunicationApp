# Real-Time Communication App

## Description
A real-time communication application using WebRTC and Socket.IO.

## Features
- Video Calling
- Audio Mute/Unmute
- Camera On/Off
- Screen Sharing
- File Sharing
- Live Chat
- Whiteboard
- Join Meeting
- Leave Meeting

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Socket.IO
- WebRTC

## Installation

npm install

## Run

node server.js

Open:

http://localhost:3000
