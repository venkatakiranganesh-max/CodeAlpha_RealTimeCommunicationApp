# Real-Time Communication App

A real-time communication web application developed using Node.js, Express.js, Socket.IO, HTML, CSS, and JavaScript.

## Features
- Real-time Chat
- Video Calling
- Screen Sharing
- Whiteboard
- File Sharing

## Technologies Used
- Node.js
- Express.js
- Socket.IO
- HTML
- CSS
- JavaScript
